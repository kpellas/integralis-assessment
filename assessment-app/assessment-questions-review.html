IT & Cyber Capability Assessment

Questions Review Document
35 Questions across 5 Capability Pillars

Assessment Overview

Purpose: This document provides all 35 assessment questions and their response options for stakeholder review.

Structure: 5 capability pillars × 7 questions each = 35 total questions

Answer Scale: Each question uses a 6-point maturity scale from "Not in place" (lowest) to "Mature/Optimised" (highest).

Scoring: Responses are converted to numerical scores: Not in place (20), Ad hoc (40), Partially implemented (60), Defined but inconsistently applied (80), Consistently applied (90), Mature/Optimised (100).
